# NewLife.Office 功能模块清单

## 总览

根据竞品分析和项目定位，NewLife.Office 功能模块按三个阶段实施。

---

## 模块清单

### M01 - 单元格样式

| 编号 | 功能 | 说明 | 阶段 | 优先级 |
|------|------|------|------|--------|
| M01-01 | CellStyle 类 | 样式模型，包含字体/填充/边框/对齐/数字格式 | 一 | 🔴高 |
| M01-02 | 字体设置 | 字体名称、大小、粗体、斜体、下划线、颜色 | 一 | 🔴高 |
| M01-03 | 填充/背景色 | 单元格背景色（纯色填充） | 一 | 🔴高 |
| M01-04 | 边框 | 四边边框样式（细/中/粗/虚线/点线/双线）与颜色 | 一 | 🔴高 |
| M01-05 | 对齐方式 | 水平对齐（左/中/右）、垂直对齐（上/中/下）、自动换行 | 一 | 🔴高 |
| M01-06 | 自定义数字格式 | 导出时指定格式（如 `#,##0.00`、`yyyy-MM-dd`） | 一 | 🟡中 |
| M01-07 | 样式管理器 | 内部组件去重、XF 缓存、styles.xml 生成引擎 | 一 | 🔴高 |

### M02 - 合并单元格

| 编号 | 功能 | 说明 | 阶段 | 优先级 |

| M02-02 | 读取合并信息 | 导入时识别合并区域 | 一 | 🟡中 |

### M03 - 冻结窗格

| M03-03 | 自定义冻结 | 同时冻结行和列 | 一 | 🟡中 |

### M04 - 自动筛选

| 编号 | 功能 | 说明 | 阶段 | 优先级 |

### M05 - 行高设置

| 编号 | 功能 | 说明 | 阶段 | 优先级 |

## 文件结构规划

```
NewLife.Office/
├── CellStyle.cs          # M01: 样式模型 + 枚举类型
├── ExcelWriter.cs        # M01-M15: 写入器（核心修改）
├── ExcelReader.cs        # M02/M07/M08: 读取器（增强）
├── ExcelTemplate.cs      # M09: 模板填充
└── NewLife.Office.csproj

XUnitTest/
├── ExcelWriterTests.cs   # 写入器单元测试
├── ExcelReaderTests.cs   # 读取器单元测试
└── IntegrationTests.cs   # 集成测试
```

## 实施顺序

1. **M01** 样式基础设施（CellStyle 类、样式管理器、styles.xml 引擎）
2. **M02** 合并单元格
3. **M03** 冻结窗格
4. **M04** 自动筛选
5. **M05** 行高设置
6. **M10** 超链接
7. **M11** 数据验证
8. **M12** 插入图片
9. **M13** 页面/打印设置
10. **M14** 工作表保护
11. **M15** 条件格式
12. **M06** 对象映射导出
13. **M07** 对象映射导入
14. **M08** DataTable 支持
15. **M09** 模板填充

# NewLife.Office 需求文档

## 1. 项目定位

NewLife.Office 是一个功能全面的 .NET 办公自动化库，覆盖 **Excel（xlsx/xls）、Word（docx/doc）、PPT（pptx/ppt）、PDF** 四大办公格式。  
**核心定位**：以 MIT 开源协议为社区提供主流办公文档的**全面操作能力**，覆盖创建、读取、编辑、模板填充、合并拆分等常见业务场景，让开发者无需依赖商业库即可完成绝大多数办公自动化任务。  
**设计原则**：零外部依赖（仅依赖 NewLife.Core），直接操作 Open XML（ZIP + XML）/ PDF 文件结构，兼顾功能全面与高性能。  
**说明**：CSV 读写操作已由 NewLife.Core 中的 `CsvFile` 类提供支持，本项目不重复实现。

## 2. Excel 功能需求

### 2.1 文件读写

- 读取和写入 xlsx 格式（Open XML，直接操作 ZIP + XML）
- 读取和写入 xls 格式（OLE2/BIFF8，兼容 Excel 97-2003）
- 多工作表支持（按名称/索引访问）
- 共享字符串处理
- 流式读写（低内存消耗，适合大文件）

### 2.2 单元格与样式

- 单元格类型自动识别（文本、数值、日期、布尔、百分比）
- 长数字保留文本（防科学计数法）、身份证号/前导零保护
- 字体（名称/大小/粗体/斜体/下划线/颜色）、背景填充色、边框
- 对齐方式（水平/垂直/自动换行）
- 自定义数字格式（`#,##0.00`、`yyyy-MM-dd` 等）
- 条件格式（数值条件高亮、数据条、色阶）
- 富文本（单元格内多种格式混排）

### 2.3 布局与布局操作

- 列宽（自动估算、手动设置）、行高（手动/自动）
- 合并单元格（读取/写入）
- 冻结窗格（行、列、行+列）
- 自动筛选
- 隐藏行/列、分组折叠

### 2.4 内容元素与高级能力

- 公式（读写，不含计算引擎）
- 超链接（URL、邮箱、工作表内部跳转）
- 数据验证（下拉列表、数值范围、日期范围）
- 图片插入与读取（drawing/media）- **图表**（柱状/折线/饼图/面积/散点图，创建与读取）
- **数据透视表**（基本创建与读取）- 对象映射导入/导出（`IEnumerable<T>` ↔ Excel，支持 `[DisplayName]`/`[Description]`）
- 模板填充（基于 xlsx 模板的占位符替换）
- 导出为 CSV（通过 NewLife.Core CsvFile 支持）

## 3. Word 功能需求

### 3.1 基础读写

- 读取/写入 docx（解析 ZIP + XML）
- 读取 doc 格式（OLE2 容器，兼容 Word 97-2003）
- 段落与样式（Heading、段落格式、粗体/斜体/颜色）
- 表格读取/写入
- 图片插入/提取
- 目录（TOC）、页眉页脚、分页符支持

### 3.2 模板填充与占位符

- 基于 docx 模板替换 `{{Key}}` 占位符，保留原有样式
- 表格区域填充（模板中标记数据区域，自动扩展行）
- 图片占位符替换

### 3.3 页面与排版

- 页面大小、方向、页边距设置
- 段落间距、行距、缩进、对齐
- 文档属性（标题/作者/主题）

### 3.4 高级功能（中低优先级）

- 书签、批注、脚注/尾注（按需实现）
- 文档保护（只读/密码）
- 对象映射导出（`IEnumerable<T>` → Word 表格）

### 3.5 格式转换与导出

- **转为 PDF（内容翻译型，纯 C# 实现）**：将 docx 段落、表格、图片映射为 PDF 基本元素，不保证像素级一致性。
  - ❗ **实现困难点**：
    1. **字体度量**：换行、分页必须测量文字宽度，纯 .NET 无零依赖方案需 SkiaSharp 或 System.Drawing；
    2. **分页算法**：范式流式布局、孤行寡行、表格跨页、浮动图片等，成熟实现需数万行代码；
    3. **全元素渲染**：页眉页脚（每页独立）、脚注尾注、目录域更新等每类需单独处理；
    4. **字体子集嵌入**： PDF 需嵌入字体子集，需 TTF/OTF 解析能力。
  - 备选方案：A）LibreOffice headless 调用（高保真，平台依赖）；B）Aspose.Words（商业许可）。
- **转为 HTML**：将 docx 结构映射为语义化 HTML（段落→`<p>`，表格→`<table>`，内联样式→CSS）。
- **转为图片**（每页渲染为 PNG/JPEG）：需要渲染引擎支持，建议通过 PDF 中转（docx→pdf→image）实现。

## 4. PPT 功能需求

### 4.1 文件读写

- 读取和创建 pptx（Open XML）
- 读取 ppt 格式（OLE2 容器，兼容 PPT 97-2003）
- 提取幻灯片文本、形状、表格、备注内容

### 4.2 幻灯片内容

- 文本框（字体/大小/颜色/粗体/斜体/对齐）
- 基本形状（矩形/圆/椭圆/箭头/线条）
- 插入图片（PNG/JPEG，指定位置与尺寸）
- 表格操作（创建/读取，支持样式）
- 超链接（文本/形状跳转）、备注页
- 图表（柱状/折线/饼图，基本生成与读取能力）

### 4.3 布局与设计

- 幻灯片母版与版式、主题/配色（基础支持）
- 背景设置（纯色/渐变/图片）
- 幻灯片大小（4:3 / 16:9 / 自定义）

### 4.4 幻灯片操作

- 添加/复制/移动/删除幻灯片
- 合并演示文稿
- 幻灯片切换（基础过渡/动画优先级低）

### 4.5 模板填充

- 占位符替换（保留原样式）、表格区域填充、图片占位符替换
- 对象映射导出（`IEnumerable<T>` → PPT 表格）

### 4.6 格式转换与导出

- **转为 PDF**：将每张幻灯片转换为 PDF 页面（可基于 PdfSharp 实现基础版）。
- **转为图片**（每张幻灯片渲染为 PNG/JPEG）：需要渲染能力，建议通过 PDF 中转实现（pptx→pdf→image）。

### 4.7 ShapeCrawler 等效形状操作

- **形状文本精确读写**：读写局部 run 级别文本格式（字体/颜色/大小/粗斜体），保留 Open XML 下层 `<a:rPr>` 结构。
- **表格单元格格式**：单元格背景色、边框、文本格式独立控制。
- **图表数据读写**：读取并修改图表系列数据（内嵌 Excel）。
- **跨文件幻灯片复制**：将幻灯片从一个 pptx 复制到另一个（含源文件样式关联）。
- **形状局位精确控制**：获取/设置形状精确位置（EMU/点）和大小。

## 5. PDF 功能需求

### 5.1 创建与生成

- 从零创建 PDF 文件（不依赖完整浏览器引擎）
- 文本排版（字体/大小/颜色/粗体/斜体）、中文字体支持
- 多页文档、表格、图片插入、页眉页脚

### 5.2 读取与提取

- 文本提取（逐页/全文）、元数据读取、页面信息
- 结构化数据提取（表格）优先级中等

### 5.3 编辑与操作

- 合并/拆分 PDF、页面旋转/删除、叠加文字/图片（盖章/水印/页码）
- 表单填充（AcroForm）及书签/大纲支持（中优先级）

### 5.4 高级功能（低优先级）

- 加密/权限控制、数字签名（需要额外依赖或商业组件）、PDF/A 合规（可选）

### 5.5 Fluent PDF 生成（QuestPDF 风格）

- **声明式布局 API**：支持 Component 嵌套组合（Row / Column / Text / Image / Table），可链式调用。
- **自动分页**：内容超出单页高度时自动扩展到新页，无需手动计算页数。
- **组件复用**：可定义可复用的布局模板块，在多个文档/页面共享。
- **组合输出**：同一布局模型根据需要输出 PDF 或 HTML（备选）。

### 5.6 底层操作（PdfSharp / MigraDoc 风格）

- **PDF 合并/拆分**：支持多文件合并、按页拆分导出。
- **低层绘图 API**：画线、矩形、弧形等基础图元，适合自定义图表/设计稿场景。
- **页面渲染为图片**：将 PDF 每页渲染为 PNG/JPEG（需引入渲染依赖，如 Docnet/SkiaSharp）。

## 6. 不做的功能

以下功能由于实现复杂度高、体积或收益有限，当前不纳入计划：

### Excel

| 功能 | 原因 |
|------|------|
| 公式计算引擎 | 实现复杂，公式读写已支持，无内置计算引擎 |
| VBA/宏 | 安全与复杂度问题 |

### Word

| 功能 | 原因 |
|------|------|
| 修订追踪 | 实现复杂、场景有限 |

### PDF

| 功能 | 原因 |
|------|------|
| HTML→PDF 高保真转换 | 需要完整渲染引擎/浏览器，体积不可控 |
| OCR 文字识别 | 需引入专业 OCR 服务，与本库定位不符 |

### PPT

| 功能 | 原因 |
|------|------|
| 完整动画时间轴 | 规范复杂，优先级低 |

## 7. 目标用户场景

1. 数据报表导出（管理后台导出用户列表、订单明细、统计报表）
2. 数据导入（批量导入用户、产品、配置等）
3. 模板报告（月报、对账单、发票等）
4. 合同/协议生成（基于 Word 模板自动填充）
5. PDF 报告生成与合并拆分
6. 演示文稿生成与模板填充（PPT）
7. 文档内容提取（PDF/PPT/Word）
8. 系统间文件交换（Excel/Word/PPT/PDF）
